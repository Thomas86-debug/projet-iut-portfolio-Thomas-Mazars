# Expérience

## Stage de 2ème année

Traitements des données et production de tableau de bords pour l’Université de Poitiers (Juin - Août 2025) :
Les universités cherchent à optimiser l’orientation et l’insertion professionnelles de leurs étudiants. Ils cherchent également à optimiser l’exploitation, comparer et valoriser les données issues de différentes sources qui est un enjeu stratégique.
L’OREVE traite des données relatives à l’insertion professionnelle des diplômés (LP, Master et Docteurs). Depuis 2024, les données d’enquêtes réalisés sur ce thème au sein de l’établissement coexistent avec celles produites par le Ministère via le dispositif InserSup, qui compile les données étudiantes issues des universités et celles concernant les emplois qui remontent du ministère du travail.
Ces deux dispositifs ont le même objectif. Cependant, la méthodologie de ces dispositifs n’est pas la même. C’est pour cela que les résultats entre ces deux dispositifs peuvent parfois différer.
A partir de ces bases, l’objectif de mon stage est de fiabiliser au maximum ces dispositifs pour pouvoir les comparer. Pour cela, je dois identifier les variables communes pour relier les deux bases de données. Pour ensuite donner concevoir des tableaux de bords qui permettra de mieux comprendre les données.
